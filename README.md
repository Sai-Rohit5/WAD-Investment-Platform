# WAD-Investment-Platform
A simple static web application that simulates a basic online investment platform. The project is fully built just using HTML.
This web application was created as part of a Web Application Development Laboratory assignment and demonstrates the use of essential HTML concepts including:

1. HTML Links
2. HTML Tables
3. HTML Lists
4. HTML Forms
5. HTML Text Formatting Tags
6. Basic Inline Styling

No JavaScript or external CSS files are used.
